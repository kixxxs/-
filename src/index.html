上传工资数据

×

CSV文件格式说明：

  • 第一行是表头，列名支持：月份、门店、性质、姓名、岗位、日薪、演出天数、合计月工资、团建费、差旅费、租金水电费
  • 必填字段：月份、门店、姓名、日薪
  • 性质：正式员工 / 兼职 / 驻场 等
  • 门店：需与系统中存在的门店名称一致
  • 月份格式：例如 2026-04
  • 合计金额 = 合计月工资 + 团建费 + 差旅费 + 租金水电费（自动计算）
  • 重要：用 Excel 编辑后请另存为 "CSV UTF-8（逗号分隔）" 格式，否则中文会乱码

上传评价数据

×

CSV文件格式说明：

  • 第一行是表头，列名支持：姓名、所在门店、演出责任心、工作情绪稳定性、团队配合度、临场应变能力、业务能力、关键标签、评价时间
  • 必填字段：姓名、所在门店
  • 5项评分为 1-10 分，综合评分自动取这5项的平均值
  • 关键标签：多个标签用逗号分隔
  • 评价时间格式：例如 2026-04-28
  • 重要：用 Excel 编辑后请另存为 "CSV UTF-8（逗号分隔）" 格式，否则中文会乱码
